# Project Brief: UMA Tools Skill Sync

## Goal
Sync missing skills from the Japanese version (umalator/uma-skill-tools) to the global version (umalator-global).

## Scope
- Identify missing skills in `umalator-global`.
- Specifically target "Ignited Spirit SPD", "Burning Spirit PWR", and "Unity Cup" skills.
- Copy definitions from `uma-skill-tools` to `umalator-global`.

## Core Components
- `uma-skill-tools`: Source of truth for JP skills.
- `umalator-global`: Target for Global skills.
